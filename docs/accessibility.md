# Accesibilidad

Objetivo: WCAG 2.2 AA práctica.

- HTML semántico con `header`, `nav`, `main`, `article`, `section` y formularios etiquetados.
- Enlace para saltar al contenido principal.
- Estados de foco visibles con `:focus-visible`.
- Errores de formulario conectados con `aria-describedby` y `aria-invalid`.
- Botones para acciones y enlaces para navegación.
- Contraste alto en tema claro y oscuro.
- Diseño responsive usable con zoom.
- Sin dependencia de color como único indicador.
- Respeto por `prefers-reduced-motion`.
